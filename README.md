## How to use this code:
### First things first, instal dependencies
`npm install`


### Development
#### Automatically build on code change
`npm start`


### Testing
#### Just once, with coverage
`npm run test`
#### Continually run tests on code change (Do this for TDD)
`npm run test-watch`


### Deployment
#### Build
`npm run build`
